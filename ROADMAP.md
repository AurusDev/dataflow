# 📍 Roadmap do DataFlow

Este roadmap define a visão de evolução do projeto **DataFlow** 🚀  

---

## ✅ Concluído
- Upload de arquivos CSV/XLSX
- Editor de dados interativo (colunas e linhas)
- Estatísticas dinâmicas
- Gráficos interativos
- Exportação em CSV, XLSX e PDF estilizado
- Deploy público via Streamlit Cloud
- README, LICENSE e CONTRIBUTING

---

## 🚧 Em andamento
- Melhorias na interface (dark/light theme refinados)
- Suporte a arquivos grandes com chunking
- Tradução dinâmica (PT/EN)

---

## 🔮 Futuro
- Exportação em JSON
- Integração com APIs externas (Google Sheets)
- Dashboards customizáveis
- Autenticação de usuário para salvar relatórios pessoais
- Templates de relatórios pré-configurados
